Liquid tuples

Liquid tuples, or liquid grooves, is a way of writting rhythms created (as far as I know) by WTF grooves, a youtube/instagram channel.

Roughly, it consist on experiments substracting or adding units from tuples, while keeping the lenght of the subunits of the tuples. One can have, for example:

  • A four note septuplet.
  • A five note triplet.
  • Nested modified tuplets (here is a website for a midi implementation. It gets weird quickly, but it sounds great when applied properly. Related to, but not the same as, Malcom Braff’s work (see here).
